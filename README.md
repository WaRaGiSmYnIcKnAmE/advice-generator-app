### What I learned

This is the first time I've tried adding animations using css:

```css
@keyframes dice-rotate {
    25% {
        -moz-transform: rotate(30deg);
        transform: rotate(30deg);
    }
    75% {
        -moz-transform: rotate(-30deg);
        transform: rotate(-30deg);
    }
}
```
